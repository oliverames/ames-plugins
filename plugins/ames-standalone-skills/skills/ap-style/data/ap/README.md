# AP Stylebook personal notes

This folder is for Oliver's personal AP Stylebook excerpts, house-specific rulings, and notes from his licensed copy of the AP Stylebook (print + apstylebook.com subscription).

## What goes here

- Verbatim entries copied from the AP Stylebook for personal reference
- Screenshots or PDF snippets from the print or online edition
- Notes on house-specific rulings where AP defers to your judgment
- Working drafts of style rulings that haven't been folded into `SKILL.md` yet

## What this folder is NOT

- Not part of the public skill — everything in this folder (except this README) is gitignored
- Not redistributed in the marketplace
- Not loaded automatically by the skill; the skill paraphrases established rules from general knowledge

## Why gitignored

The AP Stylebook is proprietary. AP retains all rights and prohibits redistribution of Stylebook material without written permission. Personal use of an excerpted copy is fine; pushing those excerpts to a public GitHub marketplace is not.

The `SKILL.md` in the parent folder carries only paraphrased rules with original examples — safe for the public marketplace. Anything verbatim lives here, private.

## Suggested file naming

- `entries-<topic>.md` — paraphrased or verbatim entries grouped by topic (e.g., `entries-numbers.md`, `entries-titles.md`)
- `updates-<year>.md` — notes on annual AP Stylebook updates (e.g., `updates-2026.md`)
- `house-rulings.md` — your personal calls where AP is silent or ambiguous

The skill's `SKILL.md` references this folder by relative path. If you want the skill to lean on specific notes, mention them by filename in the SKILL.md's "Authoritative sources" table.
