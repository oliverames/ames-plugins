# Apple Notes Mac: Keyboard Shortcuts & Gestures Reference

Source: https://support.apple.com/guide/notes/keyboard-shortcuts-and-gestures-apd46c25187e/mac

## General

| Action | Shortcut |
|--------|----------|
| Create new note | ⌘N |
| Create Quick Note | Fn-Q |
| Duplicate note | ⌘D |
| Create new folder | ⇧⌘N |
| Show main window | ⌘0 |
| List view | ⌘1 |
| Gallery view | ⌘2 |
| Show attachments | ⌘3 |
| Search all notes | ⌥⌘F |
| Navigate sections | Tab |
| Begin editing | Return |
| Return to notes list | ⌘Return |
| Print | ⌘P |
| Show/hide highlights (shared) | ⌃⌘I |
| Show/hide activity (shared) | ⌃⌘K |
| Go forward to linked note | ⌥⌘] |
| Return to previous note | ⌥⌘[ |

## Collapsible Sections

| Action | Shortcut |
|--------|----------|
| Expand section | ⌥⌘→ |
| Expand all sections | ⌥⇧⌘→ |
| Collapse section | ⌥⌘← |
| Collapse all sections | ⌥⇧⌘← |

## Table Navigation

| Action | Shortcut |
|--------|----------|
| Next row / add row | Return |
| Previous row / above table | ⇧Return |
| New paragraph in cell | ⌥Return |
| Add row above | ⌥⌘↑ |
| Add row below | ⌥⌘↓ |
| Add column right | ⌥⌘→ |
| Add column left | ⌥⌘← |
| Next cell | Tab |
| Previous cell | ⇧Tab |
| Tab in cell | ⌥Tab |
| Select cells horizontally | ⇧←/→ |
| Select cells vertically | ⇧↑/↓ |
| Select cell content | ⌘A |
| Select entire table | ⌘A, ⌘A |
