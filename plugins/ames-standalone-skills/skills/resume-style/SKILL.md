# Ames-Inspired Resume Style Guide

Industrial-strength typography for modern professionals. Based on the Ames Shovel
Company's 225-year design tradition: bold Gothic headers for authority, traditional
serifs for readability, strategic color for emphasis.

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| **Black** | `#000000` | All body text, headers, name |
| **Manufacturing Red** | `#d94a2d` | Strategic emphasis only (2-3 elements max) |
| **Heritage Gold** | `#f5a542` | Alternative accent for creative roles |
| **Precision Gray** | `#666666` | Dates, locations, metadata |
| **Rule Gray** | `#cccccc` | Optional divider lines |

**Rule**: Use red OR gold, never both. Red = bold/technical. Gold = warm/creative.

## Typography

### Headers: Franklin Gothic (or Libre Franklin)

- **Name**: Heavy/Bold Condensed, 32-36pt
- **Section Headers**: Bold, 14-16pt, ALL CAPS
- **Job Titles**: Demi/Medium, 11-12pt
- **Contact Info**: Book/Regular, 9-10pt

### Body: Crimson Text

- **Job Descriptions**: Regular, 10-11pt
- **Bullet Points**: Regular, 10-11pt
- **Dates/Locations**: Regular or Italic, 9-10pt

## Layout

- **Margins**: 0.75" all sides (never smaller)
- **Line Height**: 1.45 for body, 1.2 for headers
- **Section Spacing**: 18-24pt between sections
- **Job Spacing**: 12-14pt between positions

## Structure

### Name Block

```
OLIVER AMES
Content Strategist - Digital Marketing

oliver@email.com - 802-555-0123 - Burlington, VT - linkedin.com/in/username
```

- Name: Franklin Gothic Heavy Condensed, 36pt, Black
- Title: Franklin Gothic Book, 11pt, Black
- Contact: Franklin Gothic Book, 9pt, Precision Gray

### Section Headers

Three options (pick one per resume, stay consistent):

1. **Simple**: Franklin Gothic Bold, 16pt, ALL CAPS, +100 letter-spacing
2. **With rule**: Franklin Gothic Bold, 14pt, ALL CAPS + 0.5pt gray rule below
3. **Color accent**: Franklin Gothic Bold, 14pt, ALL CAPS + 3pt left border in red

### Experience Entry

```
Content Strategist | Company Name                           2018-Present
Burlington, Vermont

- Increased social media reach 128% through Vermont-focused content strategy
- Managed $125K annual marketing budget across 6 channels
```

- Job Title: Franklin Gothic Demi, 11pt, Black
- Company: Crimson Text Regular, 11pt, Black
- Dates: Crimson Text Regular, 10pt, Gray (right-aligned)
- Location: Crimson Text Italic, 10pt, Gray
- Bullets: Crimson Text Regular, 10.5pt, Black
- Red accent ONLY on quantifiable wins (128%, $125K, certifications)

### Skills Section

Scannable format (preferred):

```
CORE COMPETENCIES

Social Media Strategy  -  Video Production  -  SEO & PPC  -  Brand Development
```

Crimson Text Regular, 10.5pt. Bold the 3-4 most critical skills in red.

## Strategic Color Usage

### When to use Manufacturing Red

**YES**: Quantified achievements (347% growth, $2M budget), industry certifications,
critical technical skills, awards, management scope (Led team of 10).

**NO**: Subjective claims, every bullet point, entire sentences, job titles,
company names, more than 3-5 elements per page.

### When to use Heritage Gold instead

Creative/design portfolios, marketing/brand roles, hospitality/customer-facing
positions, when warmth matters more than boldness. Same 2-3 element limit applies.

## ATS Compatibility

Create **two versions**:

1. **Visual PDF**: Full design with color accents, perfect spacing
2. **ATS DOCX**: Same fonts/hierarchy, minimal formatting, no color

ATS checklist:
- Standard section headers (EXPERIENCE, EDUCATION, SKILLS)
- Left-align all text
- Single column layout
- Standard bullets
- No text in headers/footers
- No images, graphics, or text boxes
- No tables (if avoidable)

## File Naming

```
firstname-lastname-resume-2026.pdf
firstname-lastname-resume-visual.pdf
firstname-lastname-resume-ats.docx
```

Lowercase, hyphens, descriptive, dated.

## Color Accessibility

- Black on white: 21:1 contrast (WCAG AAA)
- Red #d94a2d on white: 5.2:1 contrast (WCAG AA)
- Gold #f5a542 on white: 3.1:1 (accents only, not body text)
- Gray #666666 on white: 5.7:1 contrast (WCAG AA)

## Common Mistakes

- Too many fonts (stick to Franklin Gothic + Crimson Text)
- Red everywhere (defeats strategic emphasis)
- Text below 10pt
- No white space (maintain 0.75" margins minimum)
- Both red AND gold (choose one accent)
- Centered body text (hard to scan)
- Condensed body fonts (hard to read)

## Philosophy

Bold industrial typography that commands respect. Strategic color for emphasis.
Readable serifs for extended content. Black-and-white foundation with minimal
accent color. Functional excellence over decoration.
